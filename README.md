Brief
---

Pixel Image AI App is application for a Mac or PC which uses 3rd party models to general images ready to import directly to xLights.

The goal is to make an interative prompting process to allwo for quick changes to move directly to xLights via Copy-and-Paste.


Key features
------------
- Generate pixel matrix or tree ready images from prompts
- Convert existing images to pixel friendly style for xLights
- Configurable model and API key support
